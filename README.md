# finder
졸업 작품 - 등록된 얼굴이 있는 사진을 보여주는 서비스

*src만 업로드

### 기능
* 본인 얼굴 등록
* 개인 서버에 사진 업로드, 사진에 등록된 얼굴들 별로 분류
* 공공 서버에 사진 업로드, 자신이 나온 사진만 받아볼 수 있는 기능

### 개발 동기
* 여행, 동아리, 동호회 활동 이후 받는 수백, 수천장의 사진을 분류하는 서비스의 필요성을 느낌
